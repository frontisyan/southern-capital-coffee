  <h1>Название проекта</h1>
  <p>Southern Capital — персонализированная кофейня Ростова-на-Дону с уникальным опытом заказа кофе.</p>

  <h2>Описание проекта</h2>
  <ul>
    <li>Выбор знаменитости города (Баста, Василенко, Дибров, Карнавал) для изображения на кружке.</li>
    <li>Роботизированная кастомизация кружки с выбранным стикером в реальном времени.</li>
    <li>Интерактивное участие клиента (выбор цвета крышки, оформление напитка под настроение).</li>
    <li>Проект ориентирован на молодую аудиторию, ценящую уникальные впечатления и локальные культурные коллаборации.</li>
  </ul>

  <h2>Основные функции</h2>
  <ul>
    <li>Просмотр меню напитков</li>
    <li>Выбор амбассадора для персонализации кружки</li>
    <li>Интерактивное завершение заказа</li>
    <li>Просмотр акций и специальных предложений</li>
  </ul>

  <h2>Бизнес-цели проекта</h2>
  <ul>
    <li>Создать уникальный клиентский опыт и повысить вовлечённость.</li>
    <li>Увеличить поток заказов за счёт интерактивности.</li>
    <li>Поддерживать имидж кофейни как инновационного места в Ростове.</li>
  </ul>

  <h2>Основные роли пользователей</h2>
  <ul>
    <li>Гость — выбирает напитки, амбассадора, цвет крышки и оформляет заказ.</li>
    <li>Постоянный клиент — получает историю предыдущих заказов и рекомендации.</li>
    <li>Администратор — управляет меню, обновляет стикеры знаменитостей и акции.</li>
  </ul>
